## New Machine Setup

For first-time environment setup, follow [SETUP.md](./references/SETUP.md). It covers bootstrap scripts, encoding fixes, credential setup, and OS-specific agent instructions. This only needs to be done once per machine.

On Windows, after launching the bootstrap script, read [bootstrap.log](./scripts/logs/bootstrap.log) to monitor progress and diagnose failures — do not ask the user to relay the terminal output.
